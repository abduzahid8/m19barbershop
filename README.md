# M19 Barbershop

Mobile app for booking barbershop appointments, built with **React Native (Expo)** and **Supabase**.

## Features

- Browse services and barbers
- Book, view, and manage appointments
- User authentication and profile
- Supabase-backed data (bookings, services, users)

## Tech Stack

- React Native + Expo
- React Navigation
- Supabase (auth + database)

## Getting Started

```bash
npm install
npx expo start
```
